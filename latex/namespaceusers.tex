\doxysubsection{Detailed Description}
\mbox{\hyperlink{classusers_1_1_admin}{Admin}} class is a subclass of \mbox{\hyperlink{classusers_1_1_user}{User}} class and represents an admin user in the system.

The \mbox{\hyperlink{classusers_1_1_authentication}{Authentication}} class is responsible for authenticating users, registering new customers and admins, and providing access to the corresponding customer or admin object once they are authenticated. The class maintains an Array\+List of all customers that have been registered.

The \mbox{\hyperlink{classusers_1_1_customer}{Customer}} class represents a customer of the Toffee Store. It extends the \mbox{\hyperlink{classusers_1_1_user}{User}} class and includes additional fields and methods specific to customers.
