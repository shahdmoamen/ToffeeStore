\doxysubsection{Detailed Description}
This class is responsible for managing the data of the customers, admins, orders, items and carts of the system.

The view class is responsible for handling the user interface. It provides the user with various options to register, login, browse items, view cart, place an order, add an item, close an order and logout. The class has an instance of \mbox{\hyperlink{classdata_management_1_1_data_manager}{Data\+Manager}}, Authentication and Catalog. The instance of Authentication class is used to register, login and get the current user. The instance of Catalog class is used to browse items. The class has an instance of Customer and Admin that are used to handle the user account and role respectively.
