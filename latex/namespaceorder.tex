\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classorder_1_1_cart}{Cart}} class represents the shopping cart of a customer. It contains an Array\+List of Order\+Items, the total discount, total price, and a reference to the Customer that owns the cart. It also provides methods to add and remove items from the cart, calculate the total discount, clear the cart, and checkout.

The \mbox{\hyperlink{classorder_1_1_catalog}{Catalog}} class represents a catalog of items available in the system. It provides methods for adding and removing items from the catalog, as well as retrieving all the items.

\mbox{\hyperlink{classorder_1_1_order}{Order}} class represents an order made by a customer.

An abstract class representing a payment made for an order.

The \mbox{\hyperlink{classorder_1_1_visa_payment}{Visa\+Payment}} class represents a payment made using a Visa credit card. It extends the \mbox{\hyperlink{classorder_1_1_payment}{Payment}} abstract class and implements its abstract methods.
